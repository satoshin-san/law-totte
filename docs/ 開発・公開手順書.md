# 「法令、とってきました。」 開発・公開手順書

## STEP 1: GitHubリポジトリの準備
1. GitHubにログインし、「New repository」を作成する。
   - Repository name: `law-totte` (例)
   - Public / Add a README file にチェックを入れる。
2. 作成後、緑色の「Code」ボタンからURLをコピーする。

## STEP 2: Cursorでの環境構築
1. Cursorを開き、コマンドパレット(`Cmd+Shift+P` / `Ctrl+Shift+P`)を開く。
2. `Git: Clone` を選択し、コピーしたURLを貼り付ける。
3. デスクトップなど任意のフォルダに保存して開く。
4. 以下の2つのファイルを空で作成する。
   - `app.py`
   - `requirements.txt`

## STEP 3: AIによるコーディング（ここが最重要）
1. Cursorで `Ctrl` + `I` (Composer) を開き、後述する「Cursor指示用プロンプト」を貼り付けて実行する。
2. 生成されたコードを確認し、「Accept」で保存する。
3. ターミナルで `streamlit run app.py` を実行し、動作確認する。
   - ※エラーが出たら、「このエラーを直して」とチャットに入力して修正させる。

## STEP 4: 公開（デプロイ）
1. Cursor左側の「ソース管理」アイコンから、変更をコミット＆プッシュ（Sync）する。
2. Streamlit Community Cloud にログインし、「New app」を作成。
3. GitHubリポジトリ `law-totte` を選択し、Deployする。

## STEP 5: 運用・改善
- 修正したい場合は、STEP 3に戻りCursorで修正→STEP 4のプッシュを行うだけで、自動的にWebアプリが更新される。
